Ubuntu Unity 26.04 "Resolute" Released

The Ubuntu Unity team is proud to announce the release of Ubuntu Unity 26.04, codenamed "Resolute." This release brings refreshed artwork, improved HiDPI support, and the latest Unity7 refinements on a modern Ubuntu 26.04 base. Treated as Community LTS with 3 years of support from the Ubuntu Unity team, Resolute continues our mission of keeping the classic Unity desktop alive and thriving on modern hardware.

Ubuntu Unity 24.04 LTS is here — the latest official LTS release built on the rock-solid Ubuntu 24.04 base. October 2022

Ubuntu Unity Becomes an Official Ubuntu Flavor

A historic milestone for the project: Ubuntu Unity has been accepted as an official Ubuntu flavor, starting with the 22.10 (Kinetic Kudu) release. After years of community-driven development led by Rudra Saraswat, the project now ships on Ubuntu's official cdimage servers, benefits from Ubuntu's QA infrastructure, and is recognized by the Ubuntu Technical Board alongside flavors like Kubuntu, Xubuntu, and Lubuntu.

2020

The Community Revival

What started as one teenager's passion project grew into a thriving community effort. Rudra Saraswat's Ubuntu Unity remix attracted contributors, testers, and users who refused to let Unity die. Through 2020, the project gained momentum with regular releases, growing community engagement, and the vision of one day becoming an official Ubuntu flavor — a dream that would become reality in 2022.

2020

Unity8 Rebranded as Lomiri by UBports

In a parallel effort to keep the Unity ecosystem alive, UBports — the community maintaining Ubuntu Touch — rebranded Unity8 as Lomiri in 2020. This move reflected the shell's independent evolution away from the Ubuntu project and Canonical. Lomiri continues to serve as the desktop environment for Ubuntu Touch on phones and tablets, while Ubuntu Unity focuses on the classic Unity7 desktop for PCs and laptops.

April 2017

Canonical Discontinues Unity — UBports Steps Up

In April 2017, Canonical announced the end of Unity development and the switch to GNOME 3 for future Ubuntu releases. The same year, UBports took over maintenance of Ubuntu Touch and Unity8, ensuring the mobile vision would continue. On the desktop side, Rudra Saraswat began what would become the Ubuntu Unity remix, starting a journey that would eventually lead to official flavor status.
